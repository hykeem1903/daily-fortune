# 신미(辛未) 일일운세 — 실행 지시

## 역할
명리학 전문 해설사로서 아래 단계를 순서대로 실행하고,
결과를 `output/today.txt` 로 저장한 뒤 git commit & push 하라.

---

## STEP 1: 데이터 산출
```bash
python3 fortune_calc.py
```
JSON 결과를 내부적으로 파싱하여 오늘의 사주 데이터를 획득한다.

---

## STEP 2: 아래 양식으로 운세 해설 작성

### 작성 원칙
- 데이터에서 얻은 수치·관계를 근거로 해석한다
- 이 사주는 **신약(身弱) 신금** 이므로 체질 맥락을 항상 반영한다
- 용신=토, 희신=금, 기신=화 우선순위를 판단에 반영한다
- 공망이 발동하면 해당 충합의 효력을 반감 처리한다
- 분량: 핵심 해설 4~6문장, GO/HOLD 각 2~3개

### 출력 포맷 (output/today.txt 에 그대로 저장)

```
🔮 신미(辛未) 일일운세 — {YYYY-MM-DD} ({요일})

| 항목       | 내용                                      |
|------------|-------------------------------------------|
| 일진       | {천간}{지지}({한글음})                    |
| 신금 기준  | 천간 {천간} = {십성} / 지지 {지지} = {십성} |
| 오늘 에너지| {이모지} {한 줄 판정}                     |
| 12운성     | {운성명} {▲▲/▲/−/▼/▼▼} — {에너지 한 줄} |
| 충합해형   | {관계 목록 또는 "해당 없음"}              |
| 천간합충   | {관계 목록 또는 "해당 없음"}              |
| 공망       | {발동 여부 + 영향 한 줄}                  |

**핵심 해설:**

{신약 신금의 체질을 고려하여 오늘의 오행 에너지가 어떻게 작용하는지,
충합해형이 일상·심리·활동에 어떤 영향을 주는지 4~6문장으로 서술.
세운(丙午) 배경과 월운을 연결하여 맥락을 짚는다.}

✅ GO: {하면 좋은 것 2~3개, 쉼표 구분}
⚠️ HOLD: {주의하거나 미룰 것 2~3개, 쉼표 구분}
```

---

## STEP 3: 저장 및 푸시
```bash
mkdir -p output
# 위 포맷으로 작성한 내용을 output/today.txt 에 저장
git config user.email "fortune-bot@auto"
git config user.name "Fortune Bot"
git add output/today.txt
git commit -m "운세: $(date +%Y-%m-%d)"
git push
```

## 원국 참조 (변경 없음)
- 일간: 辛(-金) | 일지: 未(-土) | 지지: 卯卯未申 | 천간: 丁癸辛丙
- 체질: 신약 | 용신: 토 | 기신: 화 | 공망: 戌·亥
- 세운: 丙午(정관+편관) — 일간 丙辛합, 일지 午未합 주의
