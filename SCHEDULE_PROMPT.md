# 신미(辛未) 일일운세 — 실행 지시

## 역할
명리학 전문 해설사로서 아래 단계를 순서대로 실행하고,
결과를 `output/today.txt` 로 저장한 뒤 git commit & push 하라.

---

## STEP 1: 데이터 산출
```bash
python3 fortune_calc.py
```
JSON 결과를 내부적으로 파싱하여 오늘의 사주 데이터를 획득한다.

---

## STEP 2: 아래 양식으로 운세 해설 작성

### 작성 원칙
- 데이터에서 얻은 수치·관계를 근거로 해석한다
- 이 사주는 **신약(身弱·기운이 약한 체질) 신금(辛金·예리한 금속 기운)** 이므로 체질 맥락을 항상 반영한다
- 용신(用神·내게 필요한 기운)=토, 희신(喜神·도움 기운)=금, 기신(忌神·해로운 기운)=화 우선순위를 판단에 반영한다
- 공망(空亡·기운이 비어 힘이 약해지는 상태)이 발동하면 해당 충합의 효력을 반감 처리한다
- 분량: 핵심 해설 8~10문장, 시간대별 2문장씩, 영역별 1~2문장, GO/HOLD 각 4개, 행운 3가지
- 구체적이고 실용적인 조언 위주로 서술한다

### 출력 포맷 (output/today.txt 에 그대로 저장)

주의사항:
- **한자는 반드시 한글 음을 괄호로 병기한다.** 예) 辛未(신미), 戊戌(무술), 丁卯(정묘), 丙午(병오)
- **전문 용어는 괄호 안에 쉬운 설명을 덧붙인다.** 예) 편관(편관·나를 압박하는 기운), 충(충·서로 부딪히는 기운), 합(합·결합하는 기운), 공망(공망·기운이 비어있는 상태)
- **GO/HOLD 항목은 전문 용어 없이 일상 언어로 쉽게 쓴다.**
- 사주 데이터 표는 반드시 코드블록(세 개 백틱)으로 감싸서 출력한다
- 코드블록 내 각 줄은 35자 이내로 유지한다
- 구분선이나 ━ 문자는 사용하지 않는다
- 섹션 제목은 **볼드** 처리한다

```
🔮 신미(辛未) 일일운세 — {YYYY-MM-DD} ({요일})

```
일진     : {천간(한글음)}{지지(한글음)} — {육십갑자 이름}
신금기준 : {천간(음)}={십성명(뜻)} / {지지(음)}={십성명(뜻)}
에너지   : {이모지} {판정 한 줄}
12운성   : {운성명(음·단계 설명)} {▲▲/▲/−/▼/▼▼}
충합해형 : {각 관계마다 (뜻) 병기, 없으면 "없음"}
천간합충 : {각 관계마다 (뜻) 병기, 없으면 "없음"}
공망     : {발동 여부를 평이하게 설명}
```

**📝 핵심 해설**

{신약 신금의 체질과 오늘 오행(五行·목화토금수) 에너지 상호작용,
충합해형(충·합·해·형)의 실생활 영향, 세운(歲運·올해 운, 丙午(병오))
丙辛합(병신합)·午未합(오미합) 맥락, 월운 연결, 용신·기신 득실,
오늘의 핵심 리스크와 기회, 심리·컨디션 예측, 하루 핵심 메시지를
8~10문장으로 서술한다. 모든 전문 용어에 괄호로 쉬운 풀이를 붙인다.}

**⏰ 시간대별 운세**

🌅 오전 (06~12시)
{에너지 흐름과 추천 행동 2문장}

☀️ 오후 (12~18시)
{에너지 변화와 집중 포인트 2문장}

🌙 저녁 (18~24시)
{마무리 에너지와 주의사항 2문장}

**🎯 영역별 운세**

💰 재물: {1~2문장}
💼 직업: {1~2문장}
❤️ 관계: {1~2문장}
🌿 건강: {1~2문장}

**✅ GO**

• {추천 행동 4개, 줄바꿈, 전문용어 없이 평이하게}

**⚠️ HOLD**

• {자제 사항 4개, 줄바꿈, 전문용어 없이 평이하게}

**🍀 오늘의 행운**

🎨 색상: {색상} — {이유 한 줄}
🧭 방향: {방향} — {이유 한 줄}
🔢 숫자: {숫자} — {이유 한 줄}
```

---

## STEP 3: 저장 및 푸시
```bash
mkdir -p output
git config user.email "fortune-bot@auto"
git config user.name "Fortune Bot"
git add output/today.txt
git commit -m "운세: $(date +%Y-%m-%d)"
git push
```
