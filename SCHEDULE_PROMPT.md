# 신미(辛未) 일일운세 — 실행 지시

## 역할
명리학 전문 해설사로서 아래 단계를 순서대로 실행하고,
결과를 `output/today.txt` 로 저장한 뒤 git commit & push 하라.

---

## STEP 1: 데이터 산출
```bash
python3 fortune_calc.py
```
JSON 결과를 내부적으로 파싱하여 오늘의 사주 데이터를 획득한다.

---

## STEP 2: 아래 양식으로 운세 해설 작성

### 작성 원칙
- 데이터에서 얻은 수치·관계를 근거로 해석한다
- 이 사주는 **신약(身弱) 신금** 이므로 체질 맥락을 항상 반영한다
- 용신=토(土), 희신=금(金), 기신=화(火) 우선순위를 판단에 반영한다
- 공망이 발동하면 해당 충합의 효력을 반감 처리한다
- 분량: 핵심 해설 8~10문장, 시간대별 2문장씩, 영역별 1~2문장, GO/HOLD 각 4개, 행운 3가지
- 구체적이고 실용적인 조언 위주로 서술한다

### 출력 포맷 (output/today.txt 에 그대로 저장)

주의사항:
- 사주 데이터 표는 반드시 코드블록(세 개 백틱)으로 감싸서 출력한다
- 코드블록 내 각 줄은 35자 이내로 유지한다
- 구분선이나 ━ 문자는 사용하지 않는다
- 섹션 제목은 **볼드** 처리한다

```
🔮 신미(辛未) 일일운세 — {YYYY-MM-DD} ({요일})

```
일진     : {천간}{지지}({한글음})
신금기준 : {천간}={십성} / {지지}={십성}
에너지   : {이모지} {판정 한 줄}
12운성   : {운성명} {▲▲/▲/−/▼/▼▼}
충합해형 : {내용 또는 없음}
천간합충 : {내용 또는 없음}
공망     : {발동여부 한 줄}
```

**📝 핵심 해설**

{신약 신금의 체질과 오늘 오행 에너지 상호작용, 충합해형의 실생활 영향,
세운(丙午) 丙辛합·午未합 맥락, 월운 연결, 용신·기신 득실,
오늘의 핵심 리스크와 기회, 심리·컨디션 예측, 하루 핵심 메시지를
8~10문장으로 서술한다.}

**⏰ 시간대별 운세**

🌅 오전 (06~12시)
{에너지 흐름과 추천 행동 2문장}

☀️ 오후 (12~18시)
{에너지 변화와 집중 포인트 2문장}

🌙 저녁 (18~24시)
{마무리 에너지와 주의사항 2문장}

**🎯 영역별 운세**

💰 재물: {1~2문장}
💼 직업: {1~2문장}
❤️ 관계: {1~2문장}
🌿 건강: {1~2문장}

**✅ GO**

• {추천 행동 4개, 줄바꿈}

**⚠️ HOLD**

• {자제 사항 4개, 줄바꿈}

**🍀 오늘의 행운**

🎨 색상: {색상} — {이유 한 줄}
🧭 방향: {방향} — {이유 한 줄}
🔢 숫자: {숫자} — {이유 한 줄}
```

## STEP 3: 저장 및 푸시
```bash
mkdir -p output
git config user.email "fortune-bot@auto"
git config user.name "Fortune Bot"
git add output/today.txt
git commit -m "운세: $(date +%Y-%m-%d)"
git push
```
